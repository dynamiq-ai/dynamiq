# Growth Playbook 🚀

A practical guide to growing Dynamiq's visibility and community.

## Project Positioning

**Dynamiq is an orchestration framework for agentic AI and LLM applications.**

### What Makes Dynamiq Different

| Feature | Dynamiq | LangChain | CrewAI | AutoGen |
|---------|---------|-----------|--------|---------|
| Workflow orchestration | ✅ Native DAG | Chain-based | Role-based | Conversation |
| Multi-agent patterns | ✅ Parallel + Sequential | Via LangGraph | Team-based | Group chat |
| Graph orchestrator | ✅ Built-in | Separate (LangGraph) | ❌ | ❌ |
| RAG pipeline | ✅ First-class | Modular | Limited | Basic |
| Memory system | ✅ Pluggable backends | Multiple options | Limited | Basic |
| Async execution | ✅ Native | Partial | Partial | ✅ |

**Core strengths:**
- Clean DAG-based workflow orchestration
- First-class RAG support with document indexing and retrieval flows
- Flexible multi-agent patterns (parallel, sequential, hierarchical)
- Graph orchestrator for custom architectures
- Production-ready with memory, connections, and prompts management
