# 🚀 GROWTH.md — Community Growth Guide for Dynamiq

> A practical roadmap to increase Dynamiq's visibility and build a thriving community.

## 📍 Positioning Analysis

**What Dynamiq Is:**
- An orchestration framework for agentic AI and LLM applications
- Specialized in RAG pipelines and multi-agent workflows
- Designed for developers building production AI systems

**Unique Differentiators:**
| Feature | vs. LangChain | vs. AutoGen | vs. CrewAI |
|---------|---------------|-------------|------------|
| Workflow orchestration | ✅ Native parallel/sequential workflows | ❌ Requires chains | ⚠️ Limited |
| Graph Orchestrator | ✅ Custom state machines | ❌ | ❌ |
| RAG Pipeline | ✅ Built-in indexing + retrieval | ✅ | ⚠️ External |
| Memory Module | ✅ Pluggable backends | ⚠️ Basic | ⚠️ Basic |
| Learning Curve | ⚡ Pythonic, intuitive | 📈 Steep | 📊 Moderate |

**Target Users:**
- AI/ML Engineers building production agents
- Teams needing multi-agent orchestration
- Developers migrating from LangChain seeking simpler abstractions

## 💬 Community Engagement

### Content That Works for Agent Frameworks

1. **Comparison Posts** (High engagement)
   - "Dynamiq vs LangChain: When to use each"
   - "Building the same agent in 5 different frameworks"
   
2. **Tutorial Series**
   - "RAG in 10 minutes with Dynamiq"
   - "From single agent to multi-agent in Dynamiq"
   - "Graph Orchestrator deep dive"

3. **Integration Showcases**
   - E2B Code Interpreter + Dynamiq
   - Pinecone + Dynamiq RAG pipeline
   - OpenAI/Claude/Local models comparison

4. **Community Spotlights**
   - Highlight user projects built with Dynamiq
   - Interview early adopters

### Engagement Tactics

- **Discord/Slack Community**: Create a space for users to share projects and get help
- **GitHub Discussions**: Enable and actively participate in Q&A
- **Weekly Office Hours**: 30-min live session for questions (builds trust)
- **Contributor Showcase**: Highlight contributors in release notes

- Test coverage improvements

- API reference: Keep in sync with releases
